캠핑장 예약 시 안내사항

예약 전 반드시 확인 부탁드립니다.

입금대기 6시간
  1. 2박 우선 예약입니다. 예약일 기준 일주일 전부터 1박 예약이 가능합니다.
  2. 캠핑장 내 사이트는 금연구역입니다. 별도로 마련된 흡연구역을 이용 바랍니다.
  3. 입실 시 쓰레기봉투는 입구 초입 사무실에서 받으실 수 있습니다. (부재 시 직접 방문하여 전달)
  4. 예약 입금 대기 시간은 6시간이며, 미입금 시 예약은 자동 취소 처리됩니다.
다른 사이트에 방해가 되지 않도록 주의 바랍니다.

보편적인 이용 규정(안전/매너)

  • 매너타임: 밤 22:00 ~ 오전 07:00 (소음·음악·고성방가 자제)
  • 화재 예방: 불멍/숯 사용 시 주변 정리, 자리는 비우지 말아주세요. 취침 전 완전 소화 필수
  • 쓰레기/분리수거: 지정 장소에 분리배출, 음식물·재활용 분리 협조
  • 전기 사용: 과도한 전열기(난로/전기그릴 등) 사용은 차단될 수 있어요 (차단 시 안전상 조치)
  • 차량 이동: 캠핑장 내 서행(보행자 우선), 야간 불필요한 이동 자제
  • 시설 이용: 공용시설은 다음 이용자를 위해 정리정돈 부탁드립니다
  • 보호자 동반: 미성년자는 보호자 동반 시 이용 가능(현장 규정 적용)
  • 반려동물: 동반 여부/규정은 사전 문의 부탁드립니다(목줄/배변 처리 필수)
  • 자연 보호: 나무/식물 훼손 금지, 연못·주변 환경 보호 협조

캠핑장 환불 안내

환불 기준은 한국소비자보호원의 숙박업 환불규정을 준용합니다.

기준환불
예약일 기준 10일전100% 환불
예약일 기준 9~7일전80% 환불
예약일 기준 6~5일전60% 환불
예약일 기준 4~3일전40% 환불
예약일 기준 2일~1일전 및 당일환불 불가
  • 예약취소/예약 변경 시 환불규정에 따릅니다. (날짜 및 사이트 변경 동일 적용)
  • 예약일은 실제 이용하는 날입니다.
  • 환불 시 이체 수수료 고객 부담이 발생할 수 있습니다.
  • 환불 신청 시 최초 예약일 기준으로 환불 처리됩니다.
  • 연박 예약 후 첫째날 입실하지 않는 경우 당일 취소 처리됩니다. (체크인 시간 외 입실 불가)
  • 우천 시에도 정상 운영되므로 당일 취소/변경은 불가합니다.

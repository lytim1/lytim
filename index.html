바비큐 시설

주원펜션의 바베큐장은 황토 돌담으로 각 동마다 분리되어 사랑하는 친구 또는 가족들과 오붓하게 즐기실 수 있는 공간입니다. 소중한 추억을 만들어보세요.

  • 비용 : 바비큐 그릴 30,000원 (그릴(석쇠), 숯 포함)
  • 숯 추가 10,000원 · 석쇠 추가 5,000원
  • 그릴만 사용 시 10,000원 (현장 결제)
  • 바비큐 준비 시간 : 저녁 6:00~9:00 (이용 12시까지)

입실 / 퇴실

  • 입실 15:00 이후
  • 퇴실 11:00 이전

IPTV / Wi‑Fi

  • 객실에서 데이터 걱정 없이 빠르고 편리한 인터넷 이용
  • 사용 가능 객실 : 전 객실

족구장

  • 이용시간/요금 : 펜션 문의
  • 구비 항목 : 공, 네트

확인 사항 / 이용 규정

  • 예약 인원 추가 시 펜션에 미리 연락 부탁드립니다.
  • 기준 인원 초과 시 1인 10,000원 추가요금 (미신고 확인 시 1인 15,000원)
  • 12개월 이상 유아/아동도 전체 인원에 포함되며 1만원 추가요금이 부과됩니다.
  • 최대 인원 초과 시 입실이 불가능할 수 있으며, 해당 사유로 환불 불가합니다.
  • 반려동물 입실 가능 펜션 외 반려동물 동반 시 입실이 거부될 수 있으며 환불 불가합니다.
  • 법적으로 청소년 혼숙 금지. 미성년자 예약/이용은 숙소 규정에 따라 결정됩니다.
  • 시설물 훼손/분실 시 손해배상 책임이 발생할 수 있습니다.
  • 객실 내 절대금연. 육류 등 냄새나는 조리 금지(위반 시 추가요금 부과 가능)
  • 과메기/생선구이류 등은 지정된 바베큐장에서 숯불 조리 및 이용 부탁드립니다.
  • 다른 이용객에게 피해가 되는 고성방가/과음/무분별한 오락은 삼가주세요.
  • 실시간 예약/중복예약 등의 사유는 안내 후 처리될 수 있습니다.
